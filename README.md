# SplitRight

SplitRight is a full-stack web application for managing shared expenses within a group. It allows users to create groups, add expenses, split costs among members, and track balances.

## Features
- User and group management  
- Expense tracking and splitting  
- Balance calculation (who owes whom)  
- Settlement tracking  
- AI-based bill scanning for extracting expense details  

## Tech Stack
- Frontend: React  
- Backend: Flask (Python)  
- Database: MySQL  

## Project Structure
- frontend/ - React application  
- backend/ - Flask APIs and business logic  
- database/ - SQL schema and setup  

## How to Run

### Backend
1. Go to the backend folder  
2. Install required dependencies  
3. Run:
   python app.py

### Frontend
1. Go to the frontend folder  
2. Install required dependencies  
3. Run:
   npm start

## Notes
- The project runs locally using localhost  
- MySQL is used for data storage
